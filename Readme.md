# Тестовое задание для Toster Media

Задачи:
- верстка лендинга по макету (мобильная версия)
- чистый JS и SCSS

Добавил от себя:
- линтеры
- сборка вебпаком

Запустить проект: `npm run start`. Проект запустится на `localhost:8080`

Собрать проект: `npm run build`.
